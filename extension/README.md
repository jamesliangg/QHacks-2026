# Java Oracle Docs Search (WXT)

A lightweight WXT web extension that lets you search Java Oracle documentation in one click.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run zip`

## Usage

1. Load the extension build output in your browser.
2. Open the popup and type a Java class or API term.
3. Hit **Search** to open Oracle docs search results.

Recent searches are saved locally in the popup.
